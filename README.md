# C-coin

数字货币 C-coin，因为是仿照 BTC 做的，且制作人的名字里都有 C 开头的，所以起了这个名。

本意是糊弄密码学大作业，至于为什么要重复造轮子是因为老师不让调包 QAQ。

现在他用来忽悠数据库大作业了 QAQ。
